3. Register Assets

üè¢

You want to create an inventory for tracking.

  1. Navigate to Buildings or Vehicles.

  4. Save the form.
Reason: Consumption data must be linked to specific assets to calculate metrics like kWh/sqm.

4. Log Consumption

üìâ

You want to record monthly energy usage.

  1. Go to Consumption page.
  2. Select the specific Asset from dropdown.
  3. Enter Year, Month, and Amount (kWh/Liters).
  4. Click Add Report.
Reason: Builds the historical data needed to visualize trends and identify anomalies.

5. Create RIOs

üí°

  1. Go to RIOs (Recommendations).
  2. Click Add New.
  3. Review the Reference Data on the right.
  4. Fill in Proposed Action and Estimates.
Reason: Formalizes opportunities for improvement (e.g., "Replace AC") into actionable items.

6. Manage Projects

üèóÔ∏è

You want to implement RIOs.
