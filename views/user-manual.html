User Manual

Guide to navigating and utilizing the LGU EEC Platform.

1. Getting Started & LGU Selection

LGU Selector: Located in the sidebar, this dropdown allows you to switch between different Local Government Units (LGUs). All data displayed (Buildings, Vehicles, Reports) is filtered based on the selected LGU.

Dashboard: The landing page provides a high-level summary of your data, including total assets, high-priority recommendations, and consumption charts for electricity and fuel.

2. Managing Assets (Buildings & Vehicles)
  • Buildings (FSBD): Navigate to the "Buildings" tab to view a list of government-owned buildings. Click "Add New Building" to register a facility. Ensure you fill in details like Floor Area and Construction Year.
  • Vehicles: Navigate to the "Vehicles" tab to manage the fleet. You can record details such as Plate Number, Make, Model, and Fuel Type.
  • Equipment (MADE): The "Equipment" tab tracks major energy-consuming devices (e.g., HVAC, Lighting) within specific buildings. You must select the parent building when adding equipment.

3. Consumption Reporting

Navigate to the Consumption page to log usage data.

  • Electricity (MECR): Select a building from the dropdown to view its history. Use the form to add a new monthly report (Year, Month, kWh consumed).
  • Fuel (MFCR): Select a vehicle to view its fuel history. Add reports including Liters consumed and Distance traveled (km).

4. Recommendations & Projects
  • Recommendations (RIO): Log identified opportunities for energy saving (e.g., "Replace LED lights"). These are linked to specific Buildings or Vehicles and categorized by Priority (High, Medium, Low).
  • Projects (PPA): Manage funded projects aimed at implementing RIOs. You can link multiple RIOs to a single Project and track Estimated vs. Actual Costs.

5. Administration

The Admin tab provides a master view of all data in the system.

  • LGUs: Create and manage Local Government Units here.
  • Data Cleanup: View lists of all entities (Buildings, Vehicles, etc.) regardless of the currently selected LGU. This is useful for finding "orphaned" records that haven't been assigned to an LGU yet.
  • Delete: Use the delete button with caution to remove erroneous entries.
Need Help?

If you encounter technical issues or need support, please contact the system administrator.
